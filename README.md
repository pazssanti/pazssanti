## 📂 Proyectos destacados

### 🔬 [Análisis exploratorio CASEN 2022](https://github.com/pazssanti/encuesta_CASEN_2022_analisis_exploratotio)
Minería de datos sobre 202.231 observaciones de la Encuesta CASEN. Análisis exploratorio, detección de outliers con z-score, imputación múltiple comparando PMM, CART y Lasso. Selección de variables con `information gain` y `gain.ratio` — nivel educacional como principal predictor del ingreso.  
`R` `mice` `FSelector` `ggplot2`

### 🤖 Machine Learning — Clasificación de Cáncer de Mama *(próximamente)*
Comparación de KNN, Regresión Logística, Árboles de Decisión, Random Forest y SVM.

### 📊 Econometría Bayesiana — Modelo Rasch Poisson (MCMC) *(próximamente)*
